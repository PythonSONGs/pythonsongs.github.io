从Windows换到Mac之后，因为Windows与Mac有好多的操作逻辑不同，快捷键也好多不同。为了让从Windows初换到Mac平台的朋友可以平顺过度，更好的在学习生活中过度从Windows过渡到Mac平台所以我这里写一写一些我从Windows过度到Mac时使用的一些小技巧。可能并不全面，因为我只是以我的使用习惯来整理这些技巧，这里只是希望能够抛砖引玉。

Mac修饰键名与对应符号

  • Command（或 Cmd）⌘
  • Shift ⇧
  • Option（或 Alt）⌥
  • Control（或 Ctrl）⌃
  • Caps Lock ⇪
  • Fn

面对APP软件的卡死

面对软件的bug或者不明原因的卡死，我们在Windows中一般使用的是救命三键（Ctrl + Alt + Delete）调出我们的任务管理器，然后关闭没有相应的软件进程。但是Mac系统我们如何操作呢？

  1. 调出“强制退出”

    在菜单栏桌面左上角苹果  标志 > 强制退出,可以调出任务窗口。

这一个操作只能关闭特定的属于当前用户的APP进程，但是如果我们需要关闭一些系统进程或者不属于当前这个用户的进程是就要用到下面这个操作了。

  1. 调出“活动监视器”

    访达 > 前往 > 应用程序 > 活动监视器

这里可以看到你系统的运行的进程，如果需要关闭每个进程就双击然后关闭即可。

返回桌面

Windows回到桌面是 win + D ，但是Mac有没有一个按键可以从直接跳回桌面呢？当然有，就是 Command + F3。但是我一般不使用这个方法。
我一般是使用

  • 快捷键方式 Command + F3
  • 触摸板(张开拇指及其他三指)
    具体可以看：左上角  标志 > 系统偏好设置 > 触控板 > 更多手势

应用间切换

  • 快捷键 Command + Tab

  • 按下F3切换
    如下图，我们一个桌面上有很多的应用。

    按下F3之后桌面就会变成这样，然后我们再用鼠标点击相应窗口即可切换

锁屏

  • 长按电源键
    长按电源键 1.5 秒，系统进入锁屏休眠状态。此时系统进入锁屏休眠的状态。

  • 组合快捷键 Command + Ctrl + Q
    按下之后屏幕回到锁屏界面，但是后台程序将继续在运行状态。将在你设定的息屏时间后才息屏休眠。当临时离开而又不想让后台程序停止时可以使用这个操作。

  • 设置屏保的方式

    桌面右键单击 > 更改桌面背景 > 屏幕保护程序 > 触发角


我是设置在右下角触发屏幕保护程序，这样我每次想锁屏时就向右下角划一下触摸板就锁屏了。

切换输入法

  • Command + Space（空格）

文本输入

  • 一般文本输入框

    • 跳转到行首 Command + 左箭头
    • 舔砖到行尾 Command + 右箭头

  • Terminal操作

    • 跳转到输入内容行首 Ctrl + A
    • 跳转到输入内容行尾 Ctrl + E

查看IP地址

  • 图形方式（无法看到网关）

    右上角WiFi标志 > 打开网络偏好设置 > 高级 > TCP/IP

  • 命令方式

    打开Terminal > 输入“ifconfig” > 输出结果里面一般第一个“en0”就是你当前的网卡。

开关机、休眠

  • 休眠
    或许许多朋友在长期使用Windows后都会被固化一个习惯—就是电脑不用时就关机。下次使用时再开机。这个其实在以前Windows系统还不太完善时，这个操作是挺好的，因为Windows系统通过开关机可以让系统重新加载，这样系统运行的一些小问题会随着机器重启而消失。但是现在的系统，比如win10之后其实系统后会在弱化一个开关机的过程，比如你win10关机其实底层的操作并不是真正的开关机，而是一个更像是休眠唤醒一样的操作，比如win10后系统的关机、启动操作两个操作所用时间，有时会比机器重启这一个操作的时间快许多。这就是因为我们win8的系统关机时系统其实执行的是一个休眠操作，系统将运行的内存数据存储到硬盘中然后切断了内存电源并重启。下次启动时一般就直接从上次保存的数据加载到内存中即完成了系统的启动过程，中间省略了软件启动、软件文件的一步一步加载的过程。但是重启的操作就是系统一步一步的把内存中运行的程序都逐个关闭，相应文档数据保存到硬盘中。下次启动时再一步一步的启动系统，启动软件，加载文件。
    这就是为什么我们在系统未关闭软件时就把电脑合盖休眠后，下次启动电脑发现上次使用的软件界面依旧还在。但是在软件为关闭就系统重启操作的话，启动后系统却只有桌面启动而已，之前运行的软件并不会出现的原因。
    以前Windows可能在休眠这个操作上还不太完善，休眠前未关闭软件，休眠后启动可能换面临软件崩溃需要重启的问题。所以大家不太经常使用休眠的操作。不过现在无论是Windows或者Mac OS系统其实都足够的优秀了，系统的休眠与唤醒也都不会再和以前一样导致系统崩溃。特别是Mac OS，因为Mac OS的系统是类UNIX系统，本身比较稳定，系统也足够优秀，所以一般使用Mac的人都不会经常将Mac关机而只是合盖休眠即可。因为Mac系统本身并不像Windows一样很多软件更新需要系统重启来生效，所以除了很底层的一些系统更新需要重启外，一般Mac不需要重启的。
    Mac的即开即用，随时待命也是Mac电脑吸引人的一大优点。想使用时开盖就能用，不用是合上盖子即可。他让你可以专注工作，而不用花费美好时间在等待系统开关机上。

  • 强制关机
    长按电源键 1.5 秒，系统进入锁屏休眠状态。此时如果继续按住电源键系统就会强制关机。
